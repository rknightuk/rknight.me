I backed [this pen Kickstarter](https://www.kickstarter.com/projects/antou/penn-minimal-by-design-built-for-versatile-writing) despite breaking my rule of no hardware Kickstarters - the company, Antou, seem to know what they're doing but we'll see. They have a lot of nice stuff on their site including [this pen rest](https://antou1010.com/product/w-paperweight-s/). The pen should be arriving in August some time.

[Pica](https://pica.joshpuckett.me/) is a new font management app that looks great and it's free. I need to get a handle on all the fonts I have.

I'm not a font designer but if I was I might need to make proofs and [Hellbox](https://hellbox.com/) is the one.

Two fascinating fonts I came across: [Datatype](https://franktisellano.github.io/datatype/index.html) is a "_variable font that turns text into charts_" and then this person [built a chess rendering engine into a font](https://medium.com/@kosalsen/how-i-built-a-chess-rendering-engine-inside-an-opentype-font-27e4a4d520e7) which is wild.

[Matt did a handwriting font](https://mattstein.com/thoughts/handwriting-font/), something I've done a couple of times to varying levels of success.

The last font and writing link is this post about [backtrack-free cursive](https://mmapped.blog/posts/52-backtrack-free-cursive). I don't write cursive (or in lowercase for that matter) but this is really interesting.

What intrigued me about [files.md](https://github.com/zakirullin/files.md) is that it uses [an API I wasn't aware of](https://developer.mozilla.org/en-US/docs/Web/API/File_System_API) to access local folders and files. I'm not dropping Obsidian any time soon but this is an API I want to try out more. 

Friend of the blog made [equinox](https://equinox.netigen.com/) to test light and dark mode at the same time which is handy.

I'd come across but not heard of [the XY problem](https://xyproblem.info/) before: "_asking about your attempted solution rather than your actual problem_". Nice to have a name for this scenario.

[P4LETTE](https://p4lette.app/#p=225d51-447a2a-b37c34-dd61a5-c8b0f6) is another colour palette generator. Can't have too many.

Daily games is _a thing_ now and I've been enjoying [4x3](https://4x3.fun). I also found [Gerrymandle](https://gerrymandle.com/) which is interesting but doesn't spark joy for me.

[Signalbox](https://www.map.signalbox.io/), a live train map, does spark joy though.

Finally this is a fun article about using [public domain image archive for art ideas](https://dhester.me/exploring-public-domain-image-archive-for-art-ideas).
