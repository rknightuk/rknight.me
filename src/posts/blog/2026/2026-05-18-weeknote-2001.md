I haven't done a weeknote in a while and this one is packed with links because I've got very behind on _all the things_ including my saved later list.

First off, I was thinking about Y2K as one does and I remembered that these keyrings were _everywhere_ in 1999. I am somewhat annoyed I missed weeknote 2000 to post this in but here we are.

![A yellow monster on a keyring with blue eyes and eating computers](https://cdn.rknight.me/site/2026/millennium-bug-keyring-front.jpg)
![The back of a yellow monster on a keyring with 2000 written on the back](https://cdn.rknight.me/site/2026/millennium-bug-keyring-back.jpg)

While I don't think all of these truly "replace" the premium ones they claim to, I'm always down for new ways to browse fonts and [Font Alternatives](https://fontalternatives.com) is pretty nice.

[Oat](https://oat.ink) is a new UI library without any crazy dependencies. 

[YouSky](https://rosswintle.uk/2026/02/yousky-my-one-person-social-network-short-version/) by Ross Wintle is a fun "one person social network".

Alex always has excellent posts and this one, [The bare minimum for syncing Git repos](https://alexwlchan.net/2026/bare-git/?ref=rss), is no different.

[Chris' AI policy](https://chriskirknielsen.com/ai-policy/#quality-and-security) is excellent.

[a genderless burger experience](https://www.youtube.com/watch?v=uqiXqxK9IRE) via [Keenan](https://gkeenan.co)

This guy [cracks spray cans and makes little scenes](https://tynerat.bigcartel.com/products) inside them.

[Incidental Music Player](https://motorcycl3.com/incidental/) looks fun.

[Disable Apple Creator Studio update popup in iWork](https://gist.github.com/Terabyte1385/dd9af4ed6d15e3f6a0c598d7a96895f0)

[Walkman.land](https://walkman.land/) is the shit I'm here for.

I was able to use [FontCrafter](https://arcade.pirillo.com/fontcrafter.html) to make a font from my handwriting, something I did many years ago with a different service. This was initally a bit finicky but it had some updates and worked perfectly after that.

[Slide Ventura](https://slideventura.com). Love it.

My buddy Ben made the [Legami Friends Pen Reference](https://legami.marmaladeofcourse.com/) inspired by the [Mildliner site](https://mildliners.rknight.me). It's so good.

[Fits on a Floppy - A Manifesto for Small Software](https://fitsonafloppy.com/). "Software should be as small as it can be. Not as a gimmick, but as a discipline.". YES.

[Ask an Astronaut](https://askanastronaut.issinrealtime.org/) takes all the questions ever asked of astronauts aboard the ISS and uses a local AI model to search and find them for you.

[ReKindle](https://rekindle.ink/) is a website as an OS for Kindles with a web browser.

[Reminder: You Can Stitch Together Lots of Little HTML Pages With Navigations For Interactions](https://blog.jim-nielsen.com/2026/small-html-pages/). Jim is very smart and this is worth your time.

[AirPodsSanity](https://github.com/Gaulomatic/AirPodsSanity) solved an apparently widespread issue where MacOS will set AirPods as the microphone input which ruins the output sound quality. 

Another app called [Paper](https://paper.pro/) but it looks nice.

[Journal Helper](https://www.journalhelper.com/) has a bunch of printable pages for your journal.

[5x5 Pixel font for tiny screens](https://maurycyz.com/projects/mcufont/)

[Delivering a dynamic hexagonal world map in just 10kb](https://calibreapp.com/blog/building-our-beloved-hex-map). This is the kind of shit I'm here for. Proper web stuff.

[KeyPad](https://bluetooth-keyboard.com/keypad/) is "a Mac app that lets you use your Mac keyboard and trackpad as a bluetooth keyboard and mouse". Smart.

[Pica, a MacOS font management app](https://pica.joshpuckett.me/)

[How To Make A Blockbuster Movie Trailer](https://www.youtube.com/watch?v=KAOdjqyG37A). I come back to this every few years because it's so good.

[Golden tickets](https://www.presentandcorrect.com/blogs/blog/golden-tickets). Old tickets with cool fonts.

[Another Day Closer to the End](https://profmusgrave.github.io/anotherday/). A webapp for doing those "X is closer to Y than Y is to now" type things.

[Mockdown — ASCII Wireframe Editor](https://www.mockdown.design/)

[Some Git Stash Commands I Find Useful](https://www.neilmacy.co.uk/blog/some-git-stash-commands/)
